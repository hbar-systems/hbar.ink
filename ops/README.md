# hbar.ink / ops

Created: 2026-04-10

Operational state for hbar.ink. Lives inside the ink repo so pending
work travels with the repo and does not pollute hbar.world.

## Files

- `TODO.md` — current pending items. Short, ruthless, dated. Finished
  items leave the file; no archive.
- `decisions.md` — non-trivial structural decisions. Create when needed.

## Conventions

- Every file in `ops/` has a creation date at the top.
- Every item in `TODO.md` has a date in brackets at the end.
- `ops/` is not for drafts or app code.
